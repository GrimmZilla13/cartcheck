# CartCheck 🛒
Know exactly how much you're spending before you get to the checkout line.

## What it does
- Scan grocery barcodes with your phone camera
- Automatically looks up the product name
- Estimates the price at your selected store using AI
- Keeps a running total as you shop

## Stores supported
Walmart, Target, Publix, Kroger, Aldi, Whole Foods, Costco, Trader Joe's
